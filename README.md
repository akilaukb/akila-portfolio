# Professional Identity and Cloud-Security Portfolio

A premium, high-performance personal portfolio showcasing expertise in **Security Engineering**, **Cloud Architecture**, and **Technical Operations**. This project serves as a comprehensive digital resume, featuring deep-dive technical projects, professional certifications, and a modern, high-conversion user interface.

## 🚀 Technical Features

- **Next.js 15 & React 19**: Utilizing the newest App Router architecture for optimal performance and SEO.
- **Framer Motion Engine**: Implemented sophisticated, stage-based scroll animations and WebGL-integrated overlays.
- **Three.js Scrolly-Canvas**: Features a dynamic, interactive space debris and star-field background that reacts to user scroll depth.
- **Custom Design System**: Built with Tailwind CSS, featuring "Cyber-Liquid Brutalism" aesthetics, glassmorphism, and premium typography (Playfair Display & Manrope).
- **Responsive Architecture**: Fully optimized for cross-browser compatibility and mobile responsiveness with touch-friendly navigation.

## 🛠️ Tech Stack

- **Framework**: Next.js (TypeScript)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Graphics**: Three.js / React Three Fiber
- **Deployment**: Vercel/Netlify optimized pipeline

## 🏗️ Core Project Highlights

- **Threat Detection SOC (Wazuh)**: Fully functional Security Operations Center deployment for real-time log analysis and endpoint protection.
- **Hybrid Cloud Networking**: Dual-VPC architecture utilizing pfSense, IPsec IKEv2 site-to-site tunnels, and OpenVPN remote access.
- **Full-Stack Enterprise Applications**: Various freelance solutions including SAAS Rental Systems and high-traffic E-commerce platforms.

## 🔧 Installation & Local Development

1. **Clone the repository:**
   ```bash
   git clone git@github.com:akilaukb/akila-portfolio.git
   cd akila-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```
   *The application will be available at `http://localhost:3000` (or `3002` if specified).*
